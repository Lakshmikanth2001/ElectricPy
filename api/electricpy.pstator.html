electricpy.pstator¶

electricpy.pstator(Pem, slip)¶

Stator Power Calculator for Induction Machine

Given the electromechanical power and the slip, this function will calculate the power related to the stator (provided or consumed).

\[P_s=\frac{P_{em}}{1-\text{slip}}\]
Parameters

  • Pem (float) – Electromechanical power in watts.

  • slip (float) – Slip factor in rad/sec.

Returns

Ps – Power related to the stator in watts.

Return type

float
